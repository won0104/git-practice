# git-practic




이 프로젝트는 git을 연ㅅㅂ하기 위한 저장소입니다.

see [Demo](https://www.google.com/}

## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirements.txt
```


### How To start

```shell
$ python main.py
```


### Dependency

버전 쓰기
- Python==3.10
- django==3.0.0
- requests==0.10.0


### Features

- 행운의 숫자 추출
- 행운의 숫자 저장
- 행운의 숫자 기반 구매
